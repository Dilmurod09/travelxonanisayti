# SEO deploy checklist for TravelX

Before publishing:

1. Check these files after deployment:
   - `index.html`
   - `robots.txt`
   - `sitemap.xml`

2. Open these URLs in the browser:
   - `https://travel-x.uz/`
   - `https://travel-x.uz/robots.txt`
   - `https://travel-x.uz/sitemap.xml`

3. Submit the site to:
   - Google Search Console: https://search.google.com/search-console
   - Yandex Webmaster: https://webmaster.yandex.com/

4. In both tools, add the sitemap URL:
   - `https://travel-x.uz/sitemap.xml`

5. Request indexing for the homepage after the site is live.

Notes:
- First-page ranking cannot be guaranteed by code alone.
- For stronger ranking, keep adding useful travel content, reviews, real contact details, Google Business Profile/Yandex Business, and backlinks from trusted local sources.
